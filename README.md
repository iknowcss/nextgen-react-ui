# Setup

1. Clone the repo
2. Install/use the right version of node
    * `nvm install v0.12.7`
    * `nvm use 0.12`
3. Install/update the node_modules
    * `npm i`
4. Install broccoli CLI
    * `npm i -g broccoli-cli`
